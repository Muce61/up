## 根因分析

请勾选**主因**类别（可多选）：

- [ ] 数据问题（来源、字段、口径）
- [ ] 代码问题（逻辑 bug、边界条件）
- [ ] 规则建模问题（交易制度、复权、费用）
- [ ] 执行问题（订单、滑点、撮合）
- [ ] 人工操作问题

请明确回答（每条都要写 是 / 否 + 证据）：

- 是否涉及**未来函数**？
- 是否涉及**幸存者偏差**？

- 是否涉及**测试集调参**？
